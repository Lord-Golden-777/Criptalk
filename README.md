CripTalk es una aplicación web de mensajería en tiempo real enfocada en privacidad, autenticación y comunicación segura.
El proyecto fue desarrollado utilizando tecnologías modernas de Python y desarrollo web para ofrecer una experiencia de chat rápida, dinámica y escalable.

La plataforma incluye:

Sistema de autenticación de usuarios
Chats en tiempo real
Gestión de contactos
Interfaz moderna y responsive
Arquitectura modular con Flask
Comunicación instantánea mediante WebSockets

Este proyecto fue creado como práctica avanzada de desarrollo backend y aplicaciones web en Python.

⚙️ Instalación

Clona el repositorio:

git clone https://github.com/Lord-Golden-777/Criptalk.git

Entra al directorio:

cd Criptalk

Instala las dependencias:

pip install -r requirements.txt

Ejecuta el proyecto:

python app.py
